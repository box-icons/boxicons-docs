# Unlocking Pro Features in the Figma

If you are a Boxicons Pro user, you can unlock the full icon library inside the Figma plugin.

To get started, click the **Pro User** button in the top-right corner of the plugin. This will open a modal asking for your API token.

## Activate Pro Access

<div className="basis-1/2 py-5" >

1. Click the **Pro User** button in the plugin.
2. Paste your API token into the input field.
3. Click **Verify**.

- 50,000+ icon variations
- All packs, styles, and weights

## Notes

- Your token is validated securely.
- You only need to enter it once per device/session.
- If your token expires or is revoked, you can re-enter a new one at any time if you have an active Pro subscription.
