# Figma Plugin

The Boxicons Figma plugin lets you search, preview, and insert Boxicons directly into your Figma designs. Itâ€™s the fastest way to use Boxicons while designing, without leaving Figma.

## Open the Plugin

1. Open any Figma file.
2. Right-click on the canvas.
3. Select **Plugins â†’ Boxicons**.

The Boxicons panel will open inside Figma.
