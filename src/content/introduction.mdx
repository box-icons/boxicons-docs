# Introduction

Welcome to the official Boxicons documentation. This guide will help you understand what Boxicons is, what’s included in each plan, and how to get the most out of the icon library in your design and development workflows.

## What is Boxicons?

Boxicons is a high-quality, modern icon library built to elevate your UI and product design.

Boxicons currently offers **5 icon packs**:

- Basic
- Filled
- Duotone
- Duotone Solid
- Duotone Mix

Each pack includes:

- **3 styles:** Regular, Rounded, Sharp
- **3 weights:** Normal, Thin, Bold

This results in **50,000+ icon variations** across all packs, giving you complete flexibility to match any design system or product aesthetic.

All icons are optimized for clarity, consistency, and performance across web and app interfaces.

## What do I get with Boxicons Pro?

> The free version is great for testing and lightweight usage. Boxicons Pro unlocks the full library and all tools.

### Free version includes

- Basic — Regular — Normal
- Filled — Regular — Normal

### Boxicons Pro includes

All remaining variations:

- All weights (Normal, Thin, Bold)
- All styles (Regular, Rounded, Sharp)
- All Duotone packs and variants

Plus premium tools and features:

- Access to **year-long updates** (new icons & new duotone releases)
- **Figma plugin** for seamless designer workflows
- **React package** for easy developer integration
- **Icon Explorer** to browse, customize, and export icons in common formats in seconds
- **Teams and Enterprise plans** with user management:
  - Teams: up to 8 users
  - Enterprise: up to 40 users
