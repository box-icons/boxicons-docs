# Get Started with Boxicons

<Image className="file-icon" src="/file.svg" alt="file" width={150} height={150} />


Getting started with Boxicons is simple and flexible, whether you are a designer working in Figma or a developer building a web or app interface. This guide walks you through the essential steps to integrate Boxicons into your workflow and start using icons right away. You’ll learn how to install Boxicons using different methods, use the Figma plugin, import icons into your code, and customize their style, weight, and appearance. Whether you’re prototyping, building a design system, or shipping a production app, Boxicons helps you stay consistent, fast, and visually polished across all your projects.
