# Usage as a Font&#x20;

You can use our icons as fonts 


### Import the CSS

Copy-paste the stylesheet `link` into your `head` to load our CSS
```html copy
<link href='https://cdn.boxicons.com/fonts/basic/boxicons.min.css' rel='stylesheet'>
```
or
```html copy
<link href='https://cdn.boxicons.com/fonts/brands/boxicons-brands.min.css' rel='stylesheet'>
```

The font is hosted on reliable CDN to serve your websites with ease
This will load boxicons font into your webpage

### PRO users
You can grab your cdn link from the usage page , where you can select which pack , style  and weight you prefer for your project

<Image src="usage.png" alt="Hello" width={928} height={677} unoptimized />

The Usage link can be found in the icon mini viewer or the avatar dropdown


### HTML 
To use an icon on your page, add a prefix `bx-` for regular icons, `bxs-` for filled variant of icons followed by the icon name and seperate class :
```html /bx bx-volleyball/ 
<i class='bx bx-volleyball'></i>
<i class='bx bxs-volleyball'></i>
```

### Starter Template
Create an HTML document and copy-paste the starter template
```html copy
<!doctype html>
<html lang='en'>
<head>
<meta charset='utf-8'>
<meta name='viewport' content='width=device-width, initial-scale=1, shrink-to-fit=no'>
<!-- Boxicons CSS -->
<link href='https://cdn.boxicons.com/fonts/basic/boxicons.min.css' rel='stylesheet'>
<link href='https://cdn.boxicons.com/fonts/brands/boxicons-brands.min.css' rel='stylesheet'>
 
<title>Hello, Boxicons!</title>
</head>
<body>
<h1>Hello, Boxicons!</h1>
<i class='bx bxs-hot'></i>
<i class='bxl bx-twitter-x'></i>
</body>
</html>
```

You can also access the SVG's from the CDN at:

```html copy
https://cdn.boxicons.com/svg/<PACK_NAME>/<ICON_NAME_WITH_PREFIX_BX_OR_BXS>.svg

<img src="https://cdn.boxicons.com/svg/basic/bx-8-ball.svg" alt="8-ball"/>
<img src="https://cdn.boxicons.com/svg/brands/bx-facebook.svg" alt="facebook"/>
```
